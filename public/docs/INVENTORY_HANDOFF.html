4. ลำดับตั้งค่า (ก่อนใช้งานจริง)

  1. สาขา (คลัง) — /admin/inventory/branches สร้างสาขา + ผูกสาขา HR (ถ้ามี)
  2. คลังสินค้า — อย่างน้อย 1 คลัง main ต่อสาขา
  3. Supplier — ผู้จำหน่ายที่ใช้รับเข้า
  4. SKU — ตั้งหน่วย, Min/Max, expiry_required, lot_tracking
  5. รับเข้า (Inbound) — สร้างใบ → อนุมัติ → ระบบสร้าง lot อัตโนมัติ
  6. ตรวจสต็อก — /admin/inventory/stock ยอดรวม + แยก lot
  7. BOM (ถ้าใช้) — /admin/inventory/bom กำหนดสูตรวัตถุดิบ

7. FEFO / Lot tracking

  • ทุกครั้งที่อนุมัติรับเข้า ระบบสร้าง inv_stock_lots พร้อม expiry (ถ้ากำหนด)
  • เบิก / ใช้ / เสียหาย / โอน ใช้ FEFO แนะนำ lot — เลือก lot เองได้จาก dropdown
  • หน้าสต็อกแสดงทั้งยอดรวมและรายการ lot คงเหลือ
  • รายงาน FEFO Override: /admin/inventory/reports/fefo-override
