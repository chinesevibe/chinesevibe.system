1. 打卡 / 下班

📍 上班打卡

  1. 点击 Rich Menu（聊天底部菜单）中的 上班 🟢 按钮
  2. 系统会请求 共享位置 — 允许以确认您在工作地点
  3. 确认打卡 → 收到 "✅ 上班成功" 消息

🔴 下班打卡

  1. 点击 Rich Menu 中的 下班 🔴 按钮
  2. 共享位置以确认
  3. 确认下班 → 收到 "🔴 下班成功" 消息，包含上班/下班时间总结
💡 提示：
  • 08:00 前打卡 → 准时 / 08:00 后 → 显示 "迟到 X 分钟"
  • 重复打卡 → 系统提示已打卡并显示时间
  • 未上班打卡就下班 → 提示先打卡
⚠ 注意： 系统使用 GPS 定位检查您是否在分店范围内。如果在区域外，系统将提示距离不足并无法打卡。
📅

2. 请假

使用方法

  1. 在聊天中输入 /leave 或 /ลา（或点击菜单"请假"）
  2. 系统会发送 请假表单 (LIFF) 链接供您填写：
    请假类型：病假 / 事假 / 年假 / 产假 等
    开始日期 - 结束日期
    原因（可选）
  3. 提交 → HR 将收到申请并进行处理
  4. 当 HR 批准或拒绝时，您将通过 LINE 收到 即时通知
💡 提示： 直接在 CNV WorkHub 聊天中输入 /leave 即可，无需寻找菜单
⏰

3. 申请加班

  1. 输入 /ot 或 /overtime（或点击菜单"申请加班"）
  2. 通过 LIFF 表单填写：
    • 加班日期
    • 开始 - 结束时间
    • 原因 / 工作内容
  3. 提交申请 → HR 审核 → 通过 LINE 通知结果
📄

4. 申请文件

  1. 输入 /doc、/document 或 /เอกสาร
  2. 通过 LIFF 填写详细信息：
    • 文件类型：工资证明 / 工作证明 / 工资单 / 其他
    • 需要的份数
    • 用途说明
  3. 提交申请 → HR 处理 → 文件准备好后通知
📢

5. 投诉 / 反馈

  1. 输入 /complaint 或 /ร้องเรียน
  2. 选择 模式：
    • 🔓 实名 — HR 知道您的身份
    • 🔒 匿名 — 匿名提交
  3. 填写标题和详细内容 → 提交
  4. HR 审核后通过 LINE 回复
📣

6. 查看公告

  1. 输入 /announce 或 /ประกาศ
  2. 系统会打开 公告门户 页面链接
  3. 查看 HR 发送给员工的所有公告
⚠ 注意： 重要公告（如假期变更、公司活动）HR 将通过 LINE 直接 推送消息 给所有员工
📦

7. 库存查询 / 收货入库

🔍 库存查询

  1. 输入 /stock 或 /สต็อก
  2. 系统打开 库存门户 页面
  3. 查看当前库存数量

📥 收货入库

  1. 输入 /inbound 或 /รับเข้า
  2. 打开 LIFF 扫描器 扫描 QR/条码
  3. 确认收货数量 → 系统自动更新库存
💡 提示： /stock 和 /inbound 命令仅对相关库存岗位的员工可用，取决于管理员分配的权限
🎧

8. 联系 HR

  1. 点击 Rich Menu 中的 联系 HR
  2. 系统会发送通知给 HR（无需打电话 — 便捷且正式）
  3. HR 会通过 LINE 直接回复您
💡 提示： 适用于非紧急事项，如咨询福利待遇、考勤记录、文件等
🌐

9. 切换语言

HR 系统支持 4 种语言，输入以下命令切换：

/th 或 /thai 或 /ไทย → 🇹🇭 ภาษาไทย
/en 或 /english → 🇬🇧 English
/zh 或 /chinese 或 /中文 → 🇨🇳 中文
/my 或 /burmese → 🇲🇲 မြန်မာ

或查看语言菜单：

/lang 或 /language 或 /ภาษา → 📋 显示语言选择菜单
🔄 LINE App 语言自动同步： 当您从 LINE 打开 LIFF（表单）时，系统会自动 读取 LINE App 的语言设置 并切换界面语言。如果您之前使用过 /zh 或 /th 等命令手动设置，系统将保持您选择的语言 — 不会随 LINE App 自动更改。
💡 提示： 菜单、通知消息和所有表单都将切换为您选择的语言！

10. 新员工注册

  1. 添加 CNV WorkHub 为 LINE 好友（关注）
  2. 系统将发送欢迎消息和 注册链接
  3. 点击链接 → 使用 LINE Login 登录
  4. 填写 员工编号 并选择 分店
  5. 提交信息 → 等待 HR 审核
  6. HR 审核通过后 → 可完全使用 CNV WorkHub 各项功能
💡 等待审核期间： 您将看到"等待 HR 审核"页面，暂时无法使用各项菜单功能，请耐心等待 HR 审核
🌐

11. 员工门户（员工网页端）

除 CNV WorkHub 外，员工还可通过浏览器访问 员工门户：

页面说明
🏠 首页仪表盘：显示今日打卡状态、剩余假期、最新公告、各功能快捷入口
👤 个人资料查看和编辑个人信息、上传头像、查看打卡二维码
⏱️ 考勤打卡记录、工时记录
📅 排班班次/轮值表
📄 请假提交请假申请，查看历史记录
📋 文件申请文件，查看处理状态
📣 公告查看 HR 发布的所有公告
💰 工资单下载历史工资单 PDF
📦 收货扫描收货（仅库存岗位员工）
📦 库存查看当前库存数量
🔑 登录方式： 使用 员工编号 + LINE Login 或通过 CNV WorkHub 登录
⚙️

12. HR / 管理员

HR 和管理员可通过 HR 管理后台 管理系统：

菜单说明
📊 仪表盘KPI 概览：总员工数、今日迟到、缺勤、待审批事项
👥 员工员工信息管理（添加、编辑、查看档案）
⏱️ 考勤查看员工考勤记录、编辑时间、审批补打卡
📅 假期审批 / 拒绝所有请假申请
⏰ 加班审批 / 拒绝所有加班申请
📄 文件管理文件申请、上传文件供员工下载
📢 投诉查看和回复投诉
📣 公告创建公告并推送至所有员工（通过 LINE 推送通知）
📦 库存管理 SKU、库存、供应商、仓库、收货、领用、报损
📊 报表导出考勤、加班、请假报表
💰 薪资设置薪资计算、运行薪资核算、生成工资单 PDF
🏢 分店管理分店、部门、职位
⚙️ 设置系统通用设置（工作时间、班次、GPS 围栏）
🔑 访问权限： 管理后台仅限具有 Owner / Manager / HR 权限的人员使用
💡 推送公告： HR 创建公告后，系统会自动通过 CNV WorkHub 推送给所有员工，无需员工手动查看
