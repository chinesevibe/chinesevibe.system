1. การเข้างาน / เลิกงาน

📍 เช็คอิน (เข้างาน)

  1. กดปุ่ม เช็คอิน 🟢 ใน Rich Menu (เมนูด้านล่างของแชท)
  2. ระบบจะขอ แชร์ตำแหน่ง — กดอนุญาตเพื่อยืนยันว่าคุณอยู่ที่ทำงาน
  3. กดยืนยันเช็คอิน → รับข้อความ "✅ เข้างานสำเร็จ"

🔴 เช็คเอาท์ (เลิกงาน)

  1. กดปุ่ม เช็คเอาท์ 🔴 ใน Rich Menu
  2. แชร์ตำแหน่งเพื่อยืนยัน
  3. กดยืนยัน → รับข้อความ "🔴 เลิกงานสำเร็จ" พร้อมสรุปเวลาเข้างาน/เลิกงาน
💡 Tips:
  • เข้าก่อน 08:00 → ตรงเวลา / หลัง 08:00 → ขึ้นว่า "สาย X นาที"
  • เช็คอินซ้ำ → ระบบแจ้งว่าเช็คอินแล้ว พร้อมเวลา
  • เช็คเอาท์โดยที่ยังไม่เช็คอิน → แจ้งเตือนให้เช็คอินก่อน
⚠ ข้อควรรู้: ระบบใช้ GPS ตรวจสอบว่าคุณอยู่ในรัศมีของสาขา หากอยู่นอกพื้นที่ ระบบจะแจ้งว่าอยู่นอกระยะและไม่สามารถเช็คอินได้
📅

2. ขอลา

วิธีใช้

  1. พิมพ์ /leave หรือ /ลา ในแชท (หรือกดเมนู ขอลา)
  2. ระบบจะส่งลิงก์ไปยัง ฟอร์มขอลา (LIFF) ให้กรอก:
    • ประเภทลา: ลาป่วย / ลากิจ / ลาพักร้อน / ลาคลอด ฯลฯ
    • วันที่เริ่มต้น – สิ้นสุด
    • เหตุผล (เพิ่มเติม)
  3. กดส่ง → HR จะได้รับเรื่องและดำเนินการ
  4. เมื่อ HR อนุมัติหรือไม่อนุมัติ คุณจะได้รับ ข้อความแจ้งผล ผ่าน LINE ทันที
💡 Tips: พิมพ์ /leave ได้เลยในแชท OA ไม่ต้องหาเมนู
⏰

3. ขอ OT

  1. พิมพ์ /ot หรือ /overtime (หรือกดเมนู ขอ OT)
  2. กรอกฟอร์มผ่าน LIFF:
    • วันที่ต้องการทำ OT
    • เวลาเริ่มต้น – สิ้นสุด
    • เหตุผล / งานที่ทำ
  3. ส่งคำขอ → HR พิจารณา → แจ้งผลทาง LINE
📄

4. ขอเอกสาร

  1. พิมพ์ /doc, /document หรือ /เอกสาร
  2. กรอกรายละเอียดผ่าน LIFF:
    • ประเภทเอกสาร: หนังสือรับรองเงินเดือน / รับรองการทำงาน / สลิปเงินเดือน / อื่น ๆ
    • จำนวนชุดที่ต้องการ
    • วัตถุประสงค์
  3. ส่งคำขอ → HR ดำเนินการจัดทำ → แจ้งเมื่อเอกสารพร้อมรับ
📢

5. ร้องเรียน / แจ้งเรื่อง

  1. พิมพ์ /complaint หรือ /ร้องเรียน
  2. เลือก โหมด ที่ต้องการ:
    • 🔓 เปิดเผยตัวตน — HR รู้ว่า你是谁
    • 🔒 ไม่เปิดเผยตัวตน — ส่งแบบ匿名
  3. กรอกหัวข้อและรายละเอียด → ส่ง
  4. HR จะตอบกลับทาง LINE เมื่อตรวจสอบเรื่อง
📣

6. ดูประกาศ

  1. พิมพ์ /announce หรือ /ประกาศ
  2. ระบบจะเปิดลิงก์ไปที่ หน้า Portal ประกาศ
  3. ดูประกาศทั้งหมดที่ HR ส่งถึงพนักงาน
⚠ หมายเหตุ: การแจ้งประกาศสำคัญ (เช่น เปลี่ยนแปลงวันหยุด, กิจกรรมบริษัท) HR จะ Push ข้อความ ไปหา พนักงานทุกคนโดยตรงทาง LINE
📦

7. เช็คสต็อก / รับเข้าสินค้า

🔍 เช็คสต็อก

  1. พิมพ์ /stock หรือ /สต็อก
  2. ระบบจะเปิดหน้า Portal เช็คสต็อก
  3. ดูจำนวนสินค้าคงคลัง ณ ปัจจุบัน

📥 รับเข้าสินค้า

  1. พิมพ์ /inbound หรือ /รับเข้า
  2. เปิด LIFF Scanner เพื่อ สแกน QR/Barcode สินค้า
  3. ยืนยันจำนวนที่รับเข้า → ระบบบันทึกสต็อกอัตโนมัติ
💡 Tips: คำสั่ง /stock และ /inbound ใช้ได้เฉพาะพนักงานที่เกี่ยวข้องกับงานสต็อกเท่านั้น ขึ้นอยู่กับสิทธิ์ที่ Admin กำหนด
🎧

8. ติดต่อ HR

  1. กด ติดต่อ HR ใน Rich Menu
  2. ระบบจะส่งข้อความแจ้งไปยัง HR (ไม่ต้องโทร — สะดวกและเป็นทางการ)
  3. HR จะตอบกลับคุณทาง LINE โดยตรง
💡 Tips: เหมาะสำหรับเรื่องที่ไม่เร่งด่วน เช่น สอบถามสวัสดิการ, การลงเวลา, เอกสารต่างๆ
🌐

9. เปลี่ยนภาษา

ระบบ HR รองรับ 4 ภาษา พิมพ์คำสั่งด้านล่างเพื่อเปลี่ยน:

/th หรือ /thai หรือ /ไทย → 🇹🇭 ภาษาไทย
/en หรือ /english → 🇬🇧 English
/zh หรือ /chinese หรือ /中文 → 🇨🇳 中文
/my หรือ /burmese → 🇲🇲 မြန်မာ

หรือกดดูเมนูภาษา:

/lang หรือ /language หรือ /ภาษา → 📋 แสดงเมนูเลือกภาษา
🔄 เปลี่ยนภาษา LINE App อัตโนมัติ: เมื่อคุณเปิด LIFF (ฟอร์มต่าง ๆ) จาก LINE ระบบจะ อ่านภาษาของ LINE App และปรับภาษาให้อัตโนมัติ ถ้าคุณพิมพ์ /en หรือ /zh มาก่อน ระบบจะคงภาษาที่คุณเลือกไว้ — ไม่เปลี่ยนตาม LINE App
💡 Tips: เมนู, ข้อความแจ้งผล, และฟอร์มทั้งหมดจะเปลี่ยนเป็นภาษาที่คุณเลือก!

🔤 Slash Commands ทั้งหมด (พิมพ์ในแชทได้เลย)

/complaint / /ร้องเรียนร้องเรียน / แจ้งเรื่อง

10. การลงทะเบียนพนักงานใหม่

  1. เพิ่ม LINE OA เป็นเพื่อน (Follow)
  2. ระบบจะส่งข้อความต้อนรับพร้อม ลิงก์ลงทะเบียน
  3. กดลิงก์ → เข้าสู่ระบบด้วย LINE Login
  4. กรอก รหัสพนักงาน และเลือก สาขา
  5. ส่งข้อมูล → รอ HR อนุมัติ
  6. เมื่อ HR อนุมัติแล้ว → ใช้บริการ LINE OA ได้เต็มรูปแบบ
💡 ระหว่างรออนุมัติ: คุณจะเห็นหน้า "รอ HR ตรวจสอบ" และยังไม่สามารถใช้เมนูต่าง ๆ ได้ รอ HR อนุมัติก่อนนะครับ
🌐

11. Employee Portal (เว็บสำหรับพนักงาน)

นอกจาก LINE OA แล้ว พนักงานยังสามารถเข้าใช้งานผ่านเว็บเบราว์เซอร์ได้ที่ Employee Portal:

หน้ารายละเอียด
🏠 หน้าแรกแดชบอร์ด แสดงสถานะเช็คอินวันนี้, วันลาคงเหลือ, ประกาศล่าสุด, ทางลัดไปฟังก์ชันต่าง ๆ
👤 โปรไฟล์ดูและแก้ไขข้อมูลส่วนตัว, อัปโหลดรูปโปรไฟล์, เช็ค QR Code สำหรับเช็คอิน
⏱️ การเข้างานประวัติการเช็คอิน-เช็คเอาท์, บันทึกเวลาทำงาน
📅 ตารางงานตารางกะ / เวรทำงาน
📄 ขอลาส่งคำขอลา และดูประวัติ
📋 เอกสารขอเอกสาร และตรวจสอบสถานะ
📣 ประกาศดูประกาศทั้งหมดจาก HR
💰 สลิปเงินเดือนดาวน์โหลดสลิปเงินเดือนย้อนหลัง
📦 รับเข้าสแกนรับสินค้า (สำหรับพนักงานสต็อก)
📦 เช็คสต็อกดูจำนวนสินค้าคงคลัง
🔑 เข้าสู่ระบบ: ใช้ รหัสพนักงาน + LINE Login หรือ Login ผ่าน LINE OA
⚙️

12. สำหรับ HR / Admin

HR และ Admin สามารถจัดการระบบผ่านหน้า HR Admin Dashboard:

เมนูรายละเอียด
📊 Dashboardภาพรวม KPI: พนักงานทั้งหมด, สายวันนี้, ขาดงาน, อนุมัติรอดำเนินการ
👥 พนักงานจัดการข้อมูลพนักงาน (เพิ่ม, แก้ไข, ดูโปรไฟล์)
⏱️ การเข้างานดูประวัติเข้างานของพนักงาน, แก้ไขเวลา, อนุมัติบันทึกเวลาย้อนหลัง
📅 วันลาอนุมัติ / ไม่อนุมัติคำขอลาทั้งหมด
⏰ OTอนุมัติ / ไม่อนุมัติคำขอ OT
📄 เอกสารจัดการคำขอเอกสาร, อัปโหลดเอกสารให้พนักงาน
📢 ร้องเรียนดูและตอบกลับเรื่องร้องเรียน
📣 ประกาศสร้างและส่งประกาศถึงพนักงานทุกคน (Push Notification ผ่าน LINE)
📦 สินค้าคงคลังจัดการ SKU, สต็อก, ซัพพลายเออร์, คลังสินค้า, รับเข้า, เบิกจ่าย, ขาด, Consumption
📊 รายงานส่งออกรายงานการเข้างาน, OT, การลา
💰 เงินเดือนตั้งค่าการคำนวณ Payroll, รัน Payroll, สร้าง PDF สลิปเงินเดือน
🏢 สาขาจัดการสาขา, แผนก, ตำแหน่ง
⚙️ ตั้งค่าตั้งค่าระบบทั่วไป (เวลาทำงาน, กะ, GEOFENCE)
🔑 การเข้าถึง: แดชบอร์ด Admin ใช้ได้เฉพาะผู้ที่มีสิทธิ์ Owner / Manager / HR เท่านั้น
💡 Push ประกาศ: เมื่อ HR สร้างประกาศ ระบบจะ Push ข้อความไปยัง LINE OA ของพนักงานทุกคนโดยอัตโนมัติ พนักงานไม่ต้องกดดูเอง
