# Responsive Admin QA Skill

## Purpose
Review and fix admin UI responsiveness before demo, QA, or production handoff.

## Breakpoints
- 375px mobile
- 430px large mobile
- 768px tablet
- 1024px laptop
- 1440px desktop

## Checklist
1. No unintended horizontal overflow.
2. Sidebar/header remain usable.
3. Tables remain readable.
4. Complex tables become cards or safe horizontal scroll on mobile.
5. Filters wrap cleanly.
6. Action buttons do not overflow.
7. Thai text does not break layout.
8. Modals fit viewport.
9. Status badges remain readable.
10. Important actions remain visible.

## Output
Report screens checked, breakpoints checked, issues found, fixes applied, and remaining limitations.
